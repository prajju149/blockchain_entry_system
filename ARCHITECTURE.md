## Cryptographic Components

### SHA256 Hashing (Block Hash)
```python
def calculate_hash(block):
    block_string = (
        str(block['index']) +
        block['timestamp'] +
        json.dumps(block['data'], sort_keys=True) +
        block['prev_hash']
    )
    return hashlib.sha256(block_string.encode()).hexdigest()

# Result: 64-character hexadecimal string
# Example: abc123def456...
```

### HMAC-SHA256 Signing (Block Signature)
```python
def sign_block(block, key):
    block_hash = block['hash']
    signature = hmac.new(
        key=key.encode(),
        msg=block_hash.encode(),
        digestmod=hashlib.sha256
    ).hexdigest()
    return signature

# Result: 64-character hexadecimal string
# Proves block author with master key
```

### Perceptual Image Hashing (Face Fingerprint)
```python
def image_fingerprint(image_path):
    # Load and resize image
    img = Image.open(image_path)
    img = img.resize((64, 64))
    
    # Convert to grayscale
    img = img.convert('L')
    
    # Get average pixel value
    avg = sum(img.getdata()) / len(img.getdata())
    
    # Create hash based on pixels above/below average
    hash_string = ''.join(
        '1' if pixel > avg else '0'
        for pixel in img.getdata()
    )
    
    return hashlib.sha256(hash_string.encode()).hexdigest()
```

### SHA256 Audio Hashing (Voice Fingerprint)
```python
def audio_fingerprint(audio_path):
    # Read first 4KB of audio file
    with open(audio_path, 'rb') as f:
        audio_data = f.read(4096)
    
    # SHA256 hash of audio data
    return hashlib.sha256(audio_data).hexdigest()
```

## Performance Considerations

### Optimization Strategies

1. **Caching Layer**
   - Cache resident list (Redis)
   - Cache blockchain blocks
   - Reduce database queries

2. **Database Indexing**
   - Index on resident_id
   - Index on timestamp
   - Index on room_number

3. **Blockchain Pruning**
   - Archive old blocks after 1 year
   - Keep full chain for last 365 days
   - Store archive separately

4. **Load Balancing**
   - Multiple Flask instances
   - Sticky sessions for consistency
   - Session storage in Redis
